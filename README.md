# summernote
# 夏日入侵企画 · 粉丝情绪能量站 💛

一个专供夏日入侵企画粉丝使用的情绪能量网站，让粉丝每天来这里抽取"情绪签"，获得来自乐队歌词、采访语录、粉丝共鸣语的能量。

## 🎯 网站特色

- **情绪盲盒**：每天抽取一句治愈的话
- **点赞评论**：与其他粉丝互动交流
- **热门便签**：查看最受欢迎的语句
- **海报分享**：一键生成小红书风格海报

## 🚀 快速开始

### 方法一：直接打开
直接用浏览器打开 `index.html` 文件即可使用。

### 方法二：使用本地服务器（推荐）

#### 使用 Python
```bash
python -m http.server 8000
```

#### 使用 Node.js
```bash
npx http-server -p 8000
```

然后在浏览器中访问 `http://localhost:8000`

## ✨ 功能说明

### 情绪盲盒
点击"摇一摇 · 今日盲盒"按钮，随机抽取一张情绪便签，包含：
- 乐队歌词摘抄
- 乐队相关语录
- 粉丝共鸣话

### 点赞与评论
- ❤️ 点赞：为喜欢的便签点赞
- 💬 评论：在便签下留言交流

### 热门便签
查看最受欢迎的便签，按点赞数排序

### 海报分享
点击分享按钮，生成小红书风格的海报图片，方便分享到社交平台

## 🎨 设计风格

- 小红书年轻风
- 夏日系、手帐风格
- 奶蓝、柠檬黄、草莓粉配色
- 大面积留白、柔和彩色卡片、圆角组件

## 💾 数据存储

目前使用 localStorage 存储用户点赞数据，刷新页面后点赞状态会保留。

## 📝 注意事项

- 所有数据存储在本地浏览器中
- 清除浏览器数据会丢失点赞记录
- 建议使用现代浏览器（Chrome、Firefox、Safari等）
